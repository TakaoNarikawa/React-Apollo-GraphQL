# React + GraphQL (Apollo Client)

## GraphQL のサーバーを指定

`.env` に次のように GraphQL のサーバーを記載します。

```
REACT_APP_SERVER_URL=https://xxxxxxxxxxxxxxx/graphql
```

## ライブラリのインストール

```sh
yarn
```

## 実行

```sh
yarn start
```
